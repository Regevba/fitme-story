# PRD: Stats / Progress Hub

> **ID:** 18.5 | **Status:** Shipped | **Priority:** P0
> **Last Updated:** 2026-04-04

---

## Purpose

Provide comprehensive analytics across 18 fitness metrics with multi-period views, interactive charts, trend analysis, and PR tracking — answering "Am I making progress?"

## Business Objective

Stats are the long-term retention driver. Users who check stats weekly have 3x higher D30 retention. The stats hub validates that training and nutrition efforts are producing results, reinforcing the habit loop.

## Target Persona(s)

- **Primary:** The Data-Driven Optimizer (deep interest in trends, body composition, PRs)
- **Secondary:** The Consistent Lifter (wants to see progressive overload confirmed)

## Current Implementation

### 18 Tracked Metrics (6 Categories)
| Category | Metrics |
|----------|---------|
| Body | Weight, body fat, lean mass, muscle mass, body water, visceral fat |
| Recovery | Readiness score, sleep, HRV, resting heart rate, VO2Max |
| Training | Training volume (kg), zone 2 minutes, steps, active calories |
| Nutrition | Protein intake, calorie intake, supplement adherence |

### Time Periods
- Daily, Weekly, Monthly, 3 Months, 6 Months

### Screens
| Screen | Description |
|--------|-------------|
| StatsView | Period selector (D/W/M/3M/6M), permanent weight + BF charts, metric carousel |
| Metric Detail | Full-page chart with header (current value, summary), interactive overlay, target line |
| ChartCard | Reusable chart wrapper with title, period label, trend delta |
| MetricCard | Single metric display with icon, value, unit, trend |

### Chart Types
- **Area + Line + Point** for trend metrics (weight, HRV, sleep, etc.)
- **Bar marks** for volume/zone2/steps
- **Interactive overlay** for tap/drag inspection with date + value tooltip
- **Target line** (dashed) for goals (weight target, BF target, protein target)
- **Y-axis bounds:** 0-100 for readiness/adherence; auto-scaled for others

### Key Features
- Coverage summary: "{N}/14 days logged" with breakdown by category
- Source badges: displays active data sources in range
- PR records: all-time best weight per exercise (Epley 1RM)
- Trend deltas with directional indicators
- Interactive chart inspection (tap/drag for individual data points)

## Kill Criteria

See "Kill criteria" row in Success Metrics above. Concrete threshold (added 2026-04-26 per CLAUDE.md rule #2): stats tab visit rate <0.5/week per active user sustained for 30 days OR average coverage <4/14 days sustained for 30 days → stats hub is considered failed as a retention surface and the chart inventory is pruned. Tagged T2 — Declared.
