# PRD: Home / Today Screen

> **ID:** 18.4 | **Status:** Shipped | **Priority:** P0
> **Last Updated:** 2026-04-04

---

## Purpose

Provide an "above-the-fold" action-first home screen that answers "What should I do today?" by surfacing readiness, goals, training CTA, and key biometric metrics — all without scrolling on iPhone.

## Business Objective

The home screen is the first thing users see on every app open. It drives session starts, biometric logging, and cross-feature engagement. A well-designed home increases DAU, reduces time-to-first-action, and surfaces the right CTA based on context.

## Target Persona(s)

- **All personas** — every user starts here

## Current Implementation

### Layout (No-Scroll, Action-First)
| Section | Description |
|---------|-------------|
| Greeting header | Animated LiveInfoStrip cycling every 5s: greeting, readiness score, supplement streak |
| Status Overview Card | Weight & body fat with target ranges, readiness context, action button |
| Goal Progress Card | Circular progress indicator, separate bars for weight/BF, on-track summary |
| Start Training Card (primary CTA) | Large circular action button, exercise type, day type menu, session estimate, recovery recommendation |
| Metrics Tile Grid | 4 tiles: HRV, Resting HR, Sleep, Steps (from LiveMetrics or manual fallback) |

### ReadinessCard (6-Page Auto-Cycling)
| Page | Content |
|------|---------|
| 0 | Readiness score — HRV, RHR, Sleep breakdown |
| 1 | Weekly training bar chart (Mon-Sun completion %) |
| 2 | Nutrition snapshot — protein progress, supplements, water |
| 3 | 7-day trends — weight, BF, HRV, sleep, volume, steps |
| 4 | Achievements — supplement streak, PRs this week, program day |
| 5 | Recovery Studio recommendation with rationale & routine steps |

### Key Features
- Responsive design (adapts to iPhone screen heights — compact, tight modes)
- Milestone modals (7/14/30/60/90-day supplement streaks, phase transitions)
- Status pulse animation on essential logging count changes
- Background gradients (orange + blue) scaling with goal progress
- Day type override menu (use today's schedule or pick any day)
- Recovery recommendation with personalized rationale
