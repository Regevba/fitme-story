# PRD: Training Tracking

> **ID:** 18.1 | **Status:** Shipped | **Priority:** P0
> **Last Updated:** 2026-04-04

---

## Purpose

Enable users to log structured strength and cardio training sessions with progressive overload tracking, session-by-session comparison, and completion analytics.

## Business Objective

Core engagement driver. Training sessions are the highest-value user action. Users who log 3+ sessions/week have 4x higher D30 retention than those who don't. Training data feeds the AI recommendation engine and readiness scoring.

## Target Persona(s)

- **Primary:** The Consistent Lifter (trains 4-6 days/week, tracks progressive overload)
- **Secondary:** The Health-Conscious Professional (trains 3-4 days/week, wants efficiency)

## Current Implementation

### Screens
| Screen | Description |
|--------|-------------|
| TrainingPlanView | Main training tab — week strip, session picker, exercise queue, floating rest timer |
| LiftLogPanel | Per-exercise set logging — weight, reps, RPE, warmup flag, previous session comparison |
| CardioLogPanel | Cardio logging — duration, HR, zone 2 detection, machine photo capture |
| SetRowView | Individual set entry — weight, reps, RPE tap bar (6-10), warmup toggle |
| FocusModeView | Full-screen single-exercise drill-down for fast data entry |
| SessionCompletionSheet | Summary — volume delta, exercises completed, PRs, duration |
| NotesEditorSheet | Session-level and exercise-level notes |

### Data Models
- **ExerciseLog** — exerciseID, sets: [SetLog], notes, computed totalVolume and bestSet
- **SetLog** — setNumber, weightKg, repsCompleted, rpe (1-10), isWarmup, notes, timestamp
- **CardioLog** — cardioType, durationMinutes, avgHeartRate, maxHeartRate, caloriesBurned, summaryImageData (encrypted JPEG)
- **DailyLog** — date, dayType, phase, exerciseLogs, cardioLogs, taskStatuses, mood, energyLevel, completionPct
- **ExerciseDefinition** — 49 exercises across 6 categories, 8 equipment types, 14 muscle groups

### Training Program
- 6-day split: Upper Push (Mon), Lower Body (Tue), Rest (Wed), Upper Pull (Thu), Full Body (Fri), Cardio Only (Sat), Rest (Sun)
- 49 exercises total: machines, free weight, calisthenics, cardio, core

### Key Features
- RPE tracking (6-10 scale per set)
- Automatic PR detection (heaviest set, excludes warmups)
- Rest timer with haptic alerts (10s and 0s)
- "Copy Last" session template
- Progressive overload suggestions (1RM estimation)
- Zone 2 HR validation for cardio
- Photo capture for cardio machine screens (encrypted)
- Volume tracking with delta vs previous session
- Milestone achievements (first workout, new PR)
