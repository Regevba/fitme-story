# PRD: Design System v2

> **ID:** 18.10 | **Status:** Shipped + Closure Sprints Complete | **Priority:** P0
> **Last Updated:** 2026-04-06

---

## Purpose

Provide a semantic, layered design system with **125 tokens**, 13+ reusable components, accessibility compliance, a CI-gated token pipeline, and a comprehensive UX foundation layer — ensuring visual consistency, accessibility, and enabling rapid feature development grounded in research-backed UX principles.

## Business Objective

A coherent design system reduces development time (reuse over rebuild), ensures brand consistency across features, and supports accessibility compliance. The token pipeline enables automated Android adaptation (Style Dictionary). CI gate prevents visual drift.

## Target Persona(s)

- **Internal:** developers, designers
- **User impact:** consistent, accessible, professional UI across all screens

## Current Implementation

### Token Architecture (4 Layers)
| Layer | File | Purpose |
|-------|------|---------|
| Source | `design-tokens/tokens.json` | Single source of truth (JSON) |
| Generated | `FitTracker/DesignSystem/DesignTokens.swift` | Auto-generated via `make tokens` (DO NOT EDIT) |
| Primitive | `FitTracker/DesignSystem/AppPalette.swift` | Raw hex colors (never used in views) |
| Semantic | `FitTracker/Services/AppTheme.swift` | Views reference ONLY this layer |

### Reusable Components
**Atomic/Molecules** (`DesignSystem/AppComponents.swift`):
- AppPickerChip, AppFilterBar, AppSheetShell, AppStatRow, AppSegmentedControl, AppProgressRing

**Composite** (`Views/Shared/AppDesignSystemComponents.swift`):
- AppCard (4 tones: standard, elevated, quiet, inverse)
- AppButton (4 hierarchies: primary, secondary, tertiary, destructive)
- AppMenuRow (icon + title + subtitle + chevron)

**Motion** (`DesignSystem/AppMotion.swift`):
- Accessibility-aware (respects isReduceMotionEnabled)
- AppSpring presets, loading animations (breathe, rotate, shimmer)

**View Modifiers** (`DesignSystem/AppViewModifiers.swift`):
- `.appEyebrowStyle()`, `.appTextRole()`, `.appCardStyle(density:)`, `.appReducedMotion()`

### CI Pipeline
- `make tokens` — regenerates DesignTokens.swift from tokens.json
- `make tokens-check` — CI gate verifying committed file matches generated output
- Style Dictionary config: `sd.config.js`

### Accessibility
- WCAG AA contrast validation (DEBUG mode)
- ColorContrastValidator: Text.primary ≈9.2:1, Text.secondary ≈5.4:1, Text.tertiary ≈4.6:1
- Motion reduced for accessibility preferences
